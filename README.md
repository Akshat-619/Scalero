# Scalero – SaaS Landing Page (Pure Tailwind CSS + JS)

A fully responsive SaaS landing page built using **pure Tailwind CSS (CDN version)** with **vanilla JavaScript** for Dark/Light mode functionality.

This project does not use:
- Bootstrap
- Any UI libraries
- Tailwind CLI
- Frameworks (React, Vue, etc.)
- Build tools

Everything runs directly from `index.html`.

---

## Features

- Responsive layout (mobile-first)
- Dark / Light mode toggle
- System theme detection (`prefers-color-scheme`)
- Theme persistence using `localStorage`
- Modern SaaS UI design
- Gradient backgrounds
- Glass-style cards
- Heroicons (SVG icons)
- Clean utility-based Tailwind structure

---

##  Tech Stack

- HTML5  
- Tailwind CSS (CDN)  
- Vanilla JavaScript  
- Heroicons (inline SVG)
